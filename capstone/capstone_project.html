Capstone Project¶

Supervised learning algorithm to classify aircraft turnarounds into different categories of towing behavior.¶

In order to successfully develop a stand allocation model, that can support many management decisions, it is crucial to also develop a preprocessing step, which is the break of the aircraft turnaround with towings. Rule of thumb rules usually provided by the allocation personnel are not accurate. That is why a supervised learning classification model might be useful.

In this project, a supervised learning algorithm will be developed, in order to predict which kind of towing behavior a turnaround will present, based on its features. A training data set, labeled data from live operations and containing a set of turnaround features will be used to train three different learning algorithms: DecisionTreeClassifier, SVC and AdaBoostClassifier. The model with the best performance, in terms of F1 score, will be chosen and further tuned.

Dummy Variables¶

The following steps are taken to get dummy variables:

1 - The original dataset is composed of 7 features + 1 label.
2 - From the 7 original features, 3 are numerical and 4 are categorical.
3 - The 4 categorical features generate 42 dummy features.
4 - The transformed dataset contains 45 (42 + 3) features + 1 label.

In [22]:
# Creating dummy variables for the four categorical features
for feature in ['acft_type', 'acft_cat', 'turnaround_type', 'turnaround_qualifier']:
    dummies = pd.get_dummies(data[feature], prefix=feature)
    data = data.join(dummies)
    data = data.drop(feature, axis=1)

# Importing the classifiers and the F1 scorer
from sklearn.tree import DecisionTreeClassifier
from sklearn.svm import SVC
from sklearn.ensemble import AdaBoostClassifier
from sklearn.metrics import f1_score

# Creating the classifier objects and generic functions to train and predict plus score
clf_A = DecisionTreeClassifier(random_state=1, max_depth=10)
clf_B = SVC(random_state=1)
clf_C = AdaBoostClassifier(random_state=1)

def train_classifier(clf, X_train, y_train):
    clf.fit(X_train, y_train)

def predict_score(clf, features, target):
    y_pred = clf.predict(features)
    return f1_score(target.values, y_pred, average='weighted')

# Getting the training and testing scores for each model and for each traiing set size
scores = pd.DataFrame(columns = ['clf_A_train', 'clf_A_test', 'clf_B_train', 'clf_B_test','clf_C_train', 'clf_C_test',], 
                            index = range(600, len(X_train), 600))

    X_train_n = X_train.sample(n=n, random_state=1)

    train_classifier(clf_A, X_train_n, y_train_n)
    scores.loc[n, 'clf_A_train'] = predict_score(clf_A, X_train_n, y_train_n)
    scores.loc[n, 'clf_A_test'] = predict_score(clf_A, X_test, y_test)
    
    train_classifier(clf_B, X_train_n, y_train_n)
    scores.loc[n, 'clf_B_train'] = predict_score(clf_B, X_train_n, y_train_n)
    scores.loc[n, 'clf_B_test'] = predict_score(clf_B, X_test, y_test)

    scores.loc[n, 'clf_C_train'] = predict_score(clf_C, X_train_n, y_train_n)
    scores.loc[n, 'clf_C_test'] = predict_score(clf_C, X_test, y_test)
